# A Glimpse Into Lumi

## Project Option
Responsive Website 

## Project Description
The final experience of the project will serve as a main hub of a personal portfolio and biography. This website can be used to see what my services and goods are. It will have many navigation icons and buttons to know a little dig into my everyday life. 

## Theme Interpretation 
I have taken the interpretation of space in two different ways: Literally and creatively. Literally in a sense that I will be using space and star themed backgrounds and icons to stylize the entire portfolio and biography page. Now I will be using the phrase, "creative space" to display my work and portfolio in a unique, interactive and creative way. 

## Intended User or Player
The intended user is not specific in any way. The main focus for the user is anyone who seems interested or relatively wanting to do work with me or use my services. It also just in general for everyone to get to know me better and get myself out there. 

## User Player Goal 
The user should be able to navigate through the website and in the end coming out with more information and details to know about my products and services for future reference. Also a great potiential for networking. 

## Main Features
As mentioned in the the wireframes, I will have a navgiation bar above that can collapse for simplicity of the mobile version of the webpage. There will be navgiation buttons and gallerys of immages you can swipe past with indicators. There will also be a generalized contact center with a message box for anyone who'd like to leave a message for me.
